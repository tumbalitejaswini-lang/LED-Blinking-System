# LED-Blinking-System
Built a series of LED blinking patterns using Arduino Uno. This project demonstrates how the behavior of LEDs can be controlled and modified through programming logic . By varying the code, different blinking sequences and patterns were achieved, providing a foundational understanding of digital output control and embedded system programming

~ Project Overview :
This project demonstrates the basic operation of controlling LEDs using an Arduino Uno. By programming different timing instructions, the LEDs can be made to blink in various patterns and sequences. The project serves as an introduction to digital output control, embedded programming, and the interaction between hardware and software.

~ Objective: 
To understand how an Arduino microcontroller controls external components and how different programming instructions can influence the behavior of LEDs.

~ Components Used :
* Arduino Uno
* LEDs - 4
* 220Ω Resistors
* Breadboard
* Jumper Wires
* USB Cable

~ Working Principle :
* The Arduino sends HIGH and LOW signals to the connected LEDs.
* A HIGH signal turns the LED ON.
* A LOW signal turns the LED OFF.
* Timing functions are used to create blinking effects.

~ Features : 
* LED ON/OFF control using Arduino
* Programmable blinking patterns
* Adjustable blinking speed
* Beginner-friendly embedded systems project

~ Learning Outcomes
* Understanding Arduino digital output pins
* Working with LEDs and resistors
* Learning basic embedded C programming
* Understanding timing and delay functions

~ Applications
* Status Indicators
* Signal Systems
* Electronic Displays
* Embedded System Prototyping
